## 📖 About

**Makaut-Mate** is a comprehensive Android app built to help MAKAUT students stay organized and access study resources with ease — completely free, no ads, no subscriptions.

Whether you're a student of **CSE, IT, ECE, AIML**, or any other branch, you'll find materials tailored specifically for your curriculum and semester.

---

## ✨ Features

- 📚 **Complete Access** — Get detailed organizers for all semesters without any cost
- 🧭 **User-Friendly Design** — Navigate through your course materials with ease
- 🌐 **Open Source** — Contribute to the project and help improve it
- 🔔 **Regular Updates** — New materials and features added frequently
- 🚫 **No Ads** — Completely free with no interruptions

## 🤝 How to Contribute?

Contributions are welcome! Here's how:

1. **Create an Issue** — Describe the bug or feature request in detail. Attach relevant screenshots or screen recordings if possible.
2. **Fork the Repository** — Fork this repo to your own GitHub account.
3. **Make Your Changes** — Fix the issue or add the feature in your forked repo.
4. **Open a Pull Request** — Title it: `Fixed #[issue-number] - [relevant-title]` or `Added Feature #[issue-number] - [relevant-title]`
5. **Wait for Review** — I'll review your PR within 24 hours. If I don't reply, feel free to email me.
