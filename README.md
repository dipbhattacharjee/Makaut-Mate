## 📖 About the App

**MAKAUT Mate** is a free, open-source Android application built exclusively for students of **Maulana Abul Kalam Azad University of Technology (MAKAUT), West Bengal.**

Whether you're preparing for exams, tracking your CGPA, or just looking for organized notes — MAKAUT Mate has everything you need, right in your pocket. No fees. No ads. Just learning.

## 📥 Installation Guide

> No Play Store needed! Follow these steps:

1. Click the **Download APK** button above
2. Open the downloaded `.apk` file on your Android phone
3. If prompted, go to **Settings → Install Unknown Apps → Allow**
4. Tap **Install** and you're done! ✅

## 🤝 How to Contribute

Contributions make this project better for every MAKAUT student! Here's how to get started:

```
1. 🍴  Fork this repository
2. 🌿  Create a new branch → git checkout -b feature/your-feature
3. ✏️   Make your changes
4. 💾  Commit → git commit -m "Added: your feature description"
5. 📤  Push → git push origin feature/your-feature
6. 🔁  Open a Pull Request with title:
        Fixed #[issue-number] - [title]
        Added Feature #[issue-number] - [title]
```

> 💡 **Tip:** Always create an issue first before working on a feature, so we can discuss it!
